# Eliminación del menú de opciones en WADI

WADI ahora funciona como un sistema de pensamiento conversacional directo.

Se eliminó:

- El bloque de opciones del tipo “Quiero entender algo mejor”
- El onboarding tipo “elige tu camino”
- Los mensajes genéricos de bienvenida

Se mantiene:

- Solo el campo de texto y la conversación directa
- Diagnóstico inmediato al primer mensaje del usuario
- Fricción lúcida, sin validación emocional

Motivo:

→ Las opciones rompían el carácter de WADI
→ Forzaban un flujo de UX genérico
→ Bloqueaban el inicio de una conversación real

Estado final:

✅ WADI ya no parece una IA de soporte.
✅ WADI ahora piensa con el usuario desde el primer mensaje.
